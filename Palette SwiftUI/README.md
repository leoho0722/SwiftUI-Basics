# Palette SwiftUI

iOS App 開發練習 − Palette SwiftUI

開發語言：Swift 5

App 最低安裝限制：iOS 15.0

開發環境：Xcode 13.1、Xcode 13.3.1

### Notes
```swift=
// TextField
.onChange(of: perform: ) // 當 TextField 值發生改變時，就會觸發

.onSubmit{} // 當 TextField 輸入完，按下 return 時，就會觸發

// Slider
onEditingChanged: {} // 當 Slider 滑動時，就會觸發
```
